# Extensions that exist:

## feedit

is really old and unmaintained. (there is the documentation?)

## xima_typo3_frontend_edit and georgringer/feediting

only adds a "Edit button" into the Frontend at the location of the tt_content element.
- not really a "frontend editor"

## frontend_editing

The would be official TYPO3 extension for frontend editing.

- but it is also unmaintained for a long time only works with TYPO3 10
- only allows editing of tt_content elements

# Summary of the State of Frontend Editing in TYPO3

In all 3 Extensions the data storage is still manually configured as tt_content elements.
